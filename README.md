# Website Technology Analyzer

## Description

A simple tool that detects basic website technologies based on URL input.

## Features

- Input website URL
- Detect basic CMS (WordPress, Shopify)
- Display results

## Tech Stack

- HTML
- CSS
- JavaScript

## Future Improvements

- Real API detection
- More technologies
- Backend integration

## Features

- Fetches live website HTML
- Detects technologies like:
  - WordPress
  - Shopify
  - React
  - Google Analytics
- Uses async API calls
